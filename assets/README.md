# HoBo POS - Central Assets

Place shared assets here:
- `logo.png` - App logo (PNG)
- `logo.svg` - App logo (SVG, fallback)
- `logo.ico` - Windows EXE & shortcut icon (generated by make_icon.py)

Use `resource_path('logo.ico')` in Python to resolve the path for both dev and PyInstaller EXE.
